HTML

Html

HTML staat voor Hypertext Markup Language. HTML gebruik je om alle elementen te plaatsen en verdelen in je website. Er zijn meer dan 100 verschillende elementen waar je gebruikt van kan maken. Deze variÃ«ren van elementen voor tekst tot elementen die een plaatje laten zien. Een HTML bestand heeft in de meeste gevallen een vrij standaard indeling waar je eerst aangeeft welk type bestand het is. Vervolgens een head elementen waar je belangrijke details van je website in verwerkt, zoals de titel van de pagina, favicon, beschrijving en een link naar je CSS stylesheet. Veel van data is allemaal niet te zien als code op de website, maar zorgt er wel voor dat de CSS juist werkt, en je site een duidelijke titel heeft.

CSS

Css

CSS staat voor Cascading Style Sheets. Met CSS maak je als het ware het uiterlijk van de website. Denk hierbij aan kleuren, lettertypes. Maar de plaats van bepaalde elementen. Om een element bijvoorbeeld rechts op de pagina te tonen kan je gebruik maken van Flexbox. CSS kan je zo simpel of moeilijk maken als je eigenlijk zelf wilt. Je kan heel simpel je hele site 1 lettertype laten gebruiken, en alles in het midden plaatsen. Maar ook een titel een andere grootte, lettertype en kleur geven als de rest van de tekst. Je kan met CSS eigenlijk oneindig veel opties veranderen, en zo je site zo uniek maken als je zelf wilt. Samengevat geef je dus je website vorm en kleur met CSS.

CSS FRAMEWORKS

Css Frameworks

Omdat CSS nogal een gedoe kan zijn om je uiteindelijke doel te bereiken zijn er Frameworks ontworpen. Dit zijn eigenlijk bouwstenen voor CSS. Hierdoor hoef je niet alles zelf te maken en bedenken maar kan je bouwsteentjes van de CSS framework gebruiken. Een voorbeeld hiervan is het veel gebruikte Bootstrap. Bootstrap kan gebruikt worden om bijvoorbeeld je site makkelijker in te delen door gebruik te maken van de column en rows bouwsteentjes. Zo hoef je niet zelf alles te positioneren met flex boxes maar kan je kolommen plaatsen om zo heel eenvoudig elementen te organiseren op een website.

W3C VALIDATION

W3C Validation

Omdat we allemaal mens zijn, en dus af een toe fouten kunnen maken bestaan er tools zoals W3C Validator. Deze website check je code van de HTML en CSS op eventuele fouten die je hebt gemaakt. Je kan een file of een website laten checken door deze website, en vervolgens krijg je een rapport. In dit rapport is te zien welke fouten er zijn en op welke regel van de code deze fout zit. Op deze manier kan je de fouten er makkelijker uithalen en je code "foutloos" krijgen. Zoals in de afbeelding hiernaast te zien, is het gelukt om na aardig wat puzzelen en fouten verbeteren zowel de HTML als CSS check uit te voeren zonder errors.
